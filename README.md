# ANIME HOUSE // V2.1 (Classic Restore)

> Plataforma de streaming de anime moderna com visual clássico e premium.

## 🌌 Visão Geral
A versão 2.1 foca em restaurar a estética original e amada do **Anime House**, mantendo a performance e o código limpo de uma aplicação moderna.

## 🛠️ Tech Stack
- **Engine:** Vanilla JS (ES6+)
- **Styling:** Tailwind CSS (via CDN)
- **Icons:** Lucide Icons
- **Data Source:** Kitsu API Protocol

## ✨ Melhorias em relação à v1
- **Navbar Original:** Retorno do logotipo e menus clássicos (Início, Temporadas, Gêneros).
- **Cards Premium:** Visual clássico com bordas arredondadas, sombras suaves e micro-animações de hover.
- **Tradução Pt-BR:** Todos os termos técnicos e tipos de anime (Série, Filme, OVA) traduzidos.
- **Busca Robusta:** Sistema de busca em tempo real com debounce e tratamento de erros.
- **Cache Local:** Carregamento ultra-rápido utilizando persistência no navegador.

## 📂 Estrutura de Arquivos
- `index.html`: Layout principal restaurado.
- `css/style.css`: Sistema de design Blue/Dark otimizado.
- `js/api.js`: Camada de comunicação com a API.
- `js/ui.js`: Renderização de componentes clássicos premium.
- `js/main.js`: Orquestrador de estado e ícones.
